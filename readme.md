## Laravel 教程 - 电商实战

开发一个电商项目，功能包括用户中心、收货地址、电商管理后台、权限管理、商品管理、商品 SKU、购物车模块、订单模块、支付模块（支付宝、微信支付）、商品评价、商品收藏、订单退款流程、优惠券模块，技术知识点包括 Laravel 中事务操作（Transaction）、支付接口调试、订单流水号生成、预加载与延迟预加载、事件和监听器、MailHog 调试邮件发送、Service 模式、自定义验证器等。
